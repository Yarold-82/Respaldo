# Paso 2: Configurar Acceso a la MAC.

Ve a **Preferencias del sistema > Compartir**.

Activa la opción **Administración remota** o **Compartir pantalla**.

En algunos casos, configura una contraseña para VNC si usas ese protocolo. (Debe ser la misma clave de inicio de sesión a la laptop)
# Paso 3: Configurar la sesión en RDM.

**Comentario:** Al presionar "click derecho" buscamos la opción de "Nueva Entrada"
![[Pasted image 20250716105426.png]]

**Comentario:** Seleccionamos la opción "ARD" (Apple Remote Desktop) para realizar la conexión por escritorio remoto a la laptop MAC.
![[Pasted image 20250716105807.png]]

**Comentario:** En esta opción completaremos los campos requeridos para realizar la conexión:

![[Pasted image 20250716110533.png]]

- **Nombre:** Para identificar el host al que deseas conectarte.
- **Anfitrión:** La dirección IP del equipo/host al que deseas conectarte.
- **Nombre de usuario:** Usar el nombre de usuario que tiene la laptop.
- **Contraseña:** Usar el nombre de usuario que tiene la laptop.
- **Puerto:** Puedes dejarlo de manera predeterminada o establecer el 5900.

Luego de establecer los parámetros, valida la conexión con la MAC.

Dentro de la oficina, debes estar conectado a la red Soluciones Laser.
En el exterior, debes tener la VPN activa.
