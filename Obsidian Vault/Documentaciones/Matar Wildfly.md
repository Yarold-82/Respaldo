Para "matar" el servicio de Java del WildFly, se hacer los siguientes pasos:

1. Ubicar "wildfly-26.1.3.Final", para remplazar los archivos.
2. Detener Wildfly.
3. Carpeta (bin) no borrar, reemplazar vecinas hermanas con el (.rar).
4. Carpeta (service) no borrar, reemplazar vecinas hermanas con el (.rar).
5. Reemplazar (standalone), el original esta en "documentos".
6. Arrancar WildFly.
7. Y generar carpeta del log.
